# Вебсайт медиатека
Данный проект реализует простой сайт с подключенный базой данных.

#Реализованно
- Регистрация;
- Авторизация;
- Личный кабинет;
- Создание сессионной корзины;
- История покупок;
- Почтовый сервер.
